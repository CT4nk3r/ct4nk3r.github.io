# ct4nk3r.github.io

Minimal Hugo-based landing page for `CT4nk3r`.

## Structure

- `hugo.toml`: site config
- `content/_index.md`: homepage content and project data
- `layouts/index.html`: minimal custom home template
- `index.html`: checked-in static page so current GitHub Pages repo still has a deployable landing page

## Local development

Install Hugo, then run:

```bash
hugo server
```

Build static output:

```bash
hugo
```

This workspace did not have Hugo installed, so template and checked-in landing page were updated together by hand.
